# Proxy Voting API

## A propos

Ceci est une API "simpliste" pour une logique métier de Proxy Voting (Démocratie Liquide).

API développée en [NodeJS](https://nodejs.org/) et [Sequelize](https://sequelize.org/)

## Lancement 

Démarrage "normal"

`npm run start`

Démarrage en mode "dévelopement"

`npm run dev`
